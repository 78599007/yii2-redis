Yii 2 Redis Cache, Session 和 ActiveRecord 
===============================================

此扩展为 Yii2 framework 提供了 [redis](http://redis.io/) key-value 存储的支持。
包含了一个 `Cache` 和 `Session` 存储句柄并且实现了 `ActiveRecord` 模式，允许
你存储 active records 到 redis 中。
